JDBC问题分析及解决思路

发表于 2020-09-29 更新于 2020-12-04 阅读次数： Valine：

问题

  • 数据库配置信息存在硬编码问题，如果需要改动需要修改代码，重新打包部署
  • 频繁的去创建和释放数据库链接，浪费资源

解决思路

  • 可以采用配置文件解决硬编码的问题。
  • 采用连接池解决拼房创建数据库连接的问题

问题

  • sql 语句，设置参数存在硬编码

解决思路

  • 配置文件解决，把 sql 封装到配置文件中

问题

  • 硬编码，需要手动封装返回结果集，如果这里有很多字段会非常繁琐。

解决思路

  • 反射、内省等技术，实现自动映射
